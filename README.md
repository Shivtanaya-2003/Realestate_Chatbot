A React-based chatbot application that helps users get information about real estate areas, compare areas, and view property price growth over time.


## Features

- **Area Analysis:** Analyze any real estate area for a quick summary.
- **Compare Areas:** Compare two or more areas with a single click.
- **Price Growth:** Visualize property price trends for a selected area.
- **Interactive Chat UI:** Chatbot interface with typing animation.
- **Persistent Chat History:** Chat history is saved in local storage.
- **View Full Information:** Clickable buttons to see detailed charts and tables without refreshing the page.



## Technologies Used

- **Frontend:** React, Bootstrap
- **Charts & Tables:** Recharts or any chart library you used
- **Backend:** Python / Django / Flask (or your backend framework)
- **Local Storage:** For saving chat history
